🛠 2. Electron 소스 (직접 빌드용)

.exe 가 없거나 직접 커스터마이징할 때. Node.js 설치된 PC 에서 npm install + npm run dist:win 으로 빌드.
📁
Dashboard_Electron_source_v5.40.zip
electron/ + src/ + scripts/ + assets/ + docs/references/ 포함
2.5 MB
📋 빌드 절차 (PowerShell, 약 15분)
  1. Node.js 설치 — nodejs.org → LTS 버전 다운로드 → 설치
  2. 소스 압축 해제 — Dashboard_Electron_source_v5.40.zip → C:\BBL_Dashboard\
  3. 의존성 설치 — cd C:\BBL_Dashboard\electron && npm install (~5분, ~150 MB 다운로드)
  4. 로컬 테스트 — npm start → Electron 창 표시되면 정상
  5. 윈도우 빌드 — npm run dist:win (~10분)
  6. 결과 확인 — electron/dist/ 폴더에 Setup.exe + portable.exe 생성
