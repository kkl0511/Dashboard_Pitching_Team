📐 좌표계 정의 (우투 기준)
• Peak Torso Counter Rot — 골반 대비 상대회전 ❌. 수평면(transverse plane)에서 몸통의 절대 반대 회전각. 부호: + 오른쪽 (3루쪽) / − 왼쪽 (홈쪽). Elite ≈ −38° (홈쪽으로 깊게 비틀림).
• Torso Forward Tilt at FP — 수직선 기준. + 앞으로 숙임 (forward) / − 뒤로 기울어짐 (lay back).
• Torso Rotation at FP — 3루 정면 자세 = 0°, 홈플레이트 정면 = 90°. + 열림 (홈쪽으로 회전 진행) / − 닫힘 (3루쪽에서 더 닫힘). Elite ≈ 2° (FP 시점에 거의 폐쇄 자세 유지).
• Torso Rotation at BR — 동일 좌표계. Elite ≈ 111° (BR 때 홈쪽 통과 후 follow-through).
• Scap Load at FP — markerless Theia 기준 부호 caveat 있음 (절댓값 권장).

⚠ CoG Decel AE 정의 불명확 — 단위 m/s, median_elite 0.02 m/s 는 max−min 으로 보기엔 너무 작음. Driveline 원 정의 ("Acceleration Equivalent" = 단위 시간당 감속량 일 가능성). 현재 본인 값은 cog_decel(=max−BR) 와 동일한 1.77 m/s 로 임시 매핑됨 → fact-check 필요.
