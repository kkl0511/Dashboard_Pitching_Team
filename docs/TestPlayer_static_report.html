📐 CoG Decel AE 정의 (Above Expected)
의미 — 실제 CoG 감속량이 구속 기반 예측치를 얼마나 초과했는지 보여주는 회귀 잔차. 구속이 빠른 투수일수록 예측 감속량도 크기 때문에, AE 는 "구속과 무관하게 평가한 block + braking 효율" 을 분리해서 본다.
산출식 — CoG Decel AE = 실제 CoG_Decel − 예측 CoG_Decel(ball_speed)
예측 회귀선 (KR cohort, n = 103, KR_pitching_processing.xlsx 가공값)
예측 = 0.0073 × ball_speed_kmh + 0.269 (m/s) · cohort 평균 ball_speed 135.5 km/h, 평균 CoG_Decel 1.26 m/s
Cohort 잔차 분포 — median ≈ +0.012 m/s, sd = 0.133 m/s · Driveline Elite reference 0.02 m/s (영문 sample), 0.26 m/s (sample player)
본인 — 실제 1.77 m/s − 예측 1.30 m/s = +0.47 m/s → 구속 대비 추가 감속량이 cohort sd 의 3.5배 (효율적 block + braking 시사)
📖 이 표 읽는 법 — 데이터 해석 가이드 (펼치기)
① 한 행은 무엇을 의미하나?
하나의 메카닉 변인이 (1) 본인은 어떤 값을 보이고, (2) Driveline 90+ mph cohort 의 중간값은 얼마이며, (3) 그 변인이 구속에 얼마나 큰 영향을 주고, (4) 본인 위치를 km/h 단위로 환산하면 elite 대비 어디쯤인지 보여줌.
② 색상 규칙
붉은 변인명 + 음의 차이 = elite 대비 부족, 보완 대상
+ 양의 차이 = elite 대비 우수, 강점 영역 (유지)
중요도 "높음" = 구속 영향 큰 핵심 변인 (우선순위 1) · "보통" · "낮음"
③ "차이 (km/h)" 의 정확한 의미
차이 = (본인 − Median Elite) / Per 1mph × 1.609
"이 변인 단독으로 봤을 때 elite 대비 km/h 환산 위치". 예: CoG Decel +7.2 km/h = 본인 CoG 감속량이 elite median 보다 0.45 m/s 큼 → 학술 회귀 기울기로 환산하면 +7.2 km/h 잠재.
⚠ ④ 흔한 오해 4가지 (반드시 주의)
(1) 모든 행의 차이를 더하면 잠재 구속 ❌
같은 모델 안의 변인끼리 서로 상관 (Trunk ω 빠르면 Pelvis ω 도 빠름). 합산 = 중복 카운트.
(2) 차이 (km/h) 는 인과 효과 ❌
상관 통계지 인과 아님. 빠른 공 → 큰 운동량 → 큰 CoG Decel 이 가능. 역인과 가능.
(3) 음의 차이 = 모두 동등 보완 ❌
중요도 높음 변인부터 우선. 낮음 변인은 부수적.
(4) 양의 차이 = 무시해도 됨 ❌
모델 점수 산정엔 +값도 반영. 강점은 유지하면서 약점만 보완해야.
✅ ⑤ 올바른 사용법
강약점 진단표로 사용 — "내 메카닉 중 어느 변인이 elite 와 차이가 크고 작은가?"
훈련 우선순위로 사용 — 큰 음의 차이 + 중요도 높음 변인 (예: Layback −4.7 km/h) 부터 drill 처방.
모델별 종합은 위쪽 "5 모델 라디아 + 표" 의 점수 (0-150) 사용 — 각 모델의 모든 변인이 가중 평균돼 산출됨.
잠재 구속은 §1 KPI 의 "메카닉 향상 기대 구속" 사용 — 5 모델 점수 모두 150 도달 시 천장.
💡 ⑥ 코칭 워크플로우 (3 단계)
Step 1. "차이 (km/h)" 컬럼 음수 + "중요도" 높음 변인을 추려냄 (보통 3–5개)
Step 2. 각 변인의 처방 (§3-5 메카닉 결함 Top 3 카드 참고) 을 다음 세션 menu 에 배치
Step 3. 3-4 주 후 재측정 → 같은 변인의 차이가 0 에 수렴 (혹은 +로) 변하면 진보. 모델 점수 (라디아) 와 잠재 구속 변동도 함께 모니터.
